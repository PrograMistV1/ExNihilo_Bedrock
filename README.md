# ExNihilo — Bedrock Port

ExNihilo is an unofficial adaptation of the popular Java mod ExNihilo: Sequentia for Minecraft Bedrock Edition.

This add-on brings the core mechanics of the original project to Bedrock Edition, allowing players to obtain resources through a structured material processing system. The central concept of the mod is the creation of a complete resource progression chain starting from basic blocks using specialized tools and gameplay mechanics.

The goal of this project is to reproduce the gameplay of the original mod as accurately as possible while adapting it to the addon system of Bedrock Edition.
