# Crucible

**Crucible** is used to melt solid materials into liquids.

It is primarily used to obtain **lava**.

## Mechanics

### Heating

The Crucible must be placed above a heat source.  
Different sources provide different melting speeds.

| Block          | Heat level |
|----------------|------------|
| Torch          | 1          |
| Redstone Torch | 1          |
| Magma Block    | 2          |
| Glowstone      | 2          |
| Shroomlight    | 2          |
| Lava           | 3          |
| Campfire       | 3          |
| Soul Campfire  | 3          |
| Fire           | 4          |
| Soul Fire      | 4          |
