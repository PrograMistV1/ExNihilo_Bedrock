# ExNihilo Bedrock

ExNihilo Bedrock is a port of the classic ExNihilo: Sequentia for Minecraft Bedrock Edition. The add-on is built around
the "from nothing" progression, where the player gradually gains resources through core processing mechanics.

## Main Idea

The player starts with almost no resources and builds up infrastructure through:

- sifting blocks
- processing materials
- using basic tools
- creating production chains

## Mechanics

- crook for obtaining silkworms and string
- barrel for composting, collecting rainwater, storing and transforming liquids
- sieve for sifting blocks
- hammer for crushing blocks
- crucible for melting materials and creating liquids
